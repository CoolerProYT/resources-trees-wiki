# Sapling Types

As this version doesn't support adding custom sapling type, there will be only built-in sapling type can be used. But it is also a good news, as no custom sapling type, there is lesser recipe need to be added for a new resources type.

## Saplings

Saplings item are used to define in recipe, as resources sapling crafting output or tree simulator input. All the saplings are data driven, meaning that all resources type share one same sapling.

Here are a list of available saplings:

| Sapling Type | Sapling ID                                  | Leaves ID                                  |
| ------------ | ------------------------------------------- | ------------------------------------------ |
| Oak          | `resourcestrees:resources_oak_sapling`      | `resourcestrees:resources_oak_leaves`      |
| Spruce       | `resourcestrees:resources_spruce_sapling`   | `resourcestrees:resources_spruce_leaves`   |
| Birch        | `resourcestrees:resources_birch_sapling`    | `resourcestrees:resources_birch_leaves`    |
| Jungle       | `resourcestrees:resources_jungle_sapling`   | `resourcestrees:resources_jungle_leaves`   |
| Acacia       | `resourcestrees:resources_acacia_sapling`   | `resourcestrees:resources_acacia_leaves`   |
| Dark Oak     | `resourcestrees:resources_dark_oak_sapling` | `resourcestrees:resources_dark_oak_leaves` |
| Cherry       | `resourcestrees:resources_cherry_sapling`   | `resourcestrees:resources_cherry_leaves`   |
| Pale Oak     | `resourcestrees:resources_pale_oak_sapling` | `resourcestrees:resources_pale_oak_leaves` |
